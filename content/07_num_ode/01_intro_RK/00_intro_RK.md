# Einführung in numerische Integrationsverfahren von ODEs

In diesen Übungsaufgaben werden implizite und explizite Runge-Kutta-Verfahren auf der Basis der numerischen Integration bekannter Funktionen eingeführt. In fünf Aufgaben lernen Sie

1. Explizite Integrationsverfahren von gewöhnlichen Differentialgleichungen
2. Implizite Integrationsverfahren von gewöhnlichen Differentialgleichungen
3. Grundkenntnisse der adaptiven Schrittweitensteuerung
4. Lösen von mehrdimensionalen gewöhnlichen Differentialgleichungen
5. Bewertung einiger grundlegender Integrationmethoden
